Mergify offers badges that you can use in your project to inform contributors that your repository is managed by Mergify.

Several styles are available, both on svg and in png formats.

style=flat:

style=flat-square:

style=popout:

style=popout-square:

style=plastic:

style=for-the-badge:

style=social:

In a README.md:

# My project name [![Mergify Status][mergify-status]][mergify]

[mergify]: https://mergify.io
[mergify-status]: https://img.shields.io/endpoint.svg?url=https://gh.mergify.io/badges/<owner>/<repo>&style=flat

In a README.rst:

===============
My project name
===============

.. image:: https://img.shields.io/endpoint.svg?url=https://gh.mergify.io/badges/<owner>/<repo>&style=flat
   :target: https://mergify.io
   :alt: Mergify Status

Note: <owner> and <repo> are placeholders which have to be adapted to the repo. For example: https://gh.mergify.io/badges/Mergifyio/mergify-engine?style=cut
