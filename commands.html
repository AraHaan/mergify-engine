Commands¶

Actions can be run directly by commenting on a pull request. The syntax is:

@Mergifyio [parameters …]

The list of available commands is listed below, with their parameters:

rebase¶

Will run the rebase action.

Syntax

@Mergifyio rebase

Example

@Mergifyio rebase

backport¶

Will run the backport action.

Syntax

@Mergifyio backport <branch name> <branch name 2> …

Example

@Mergifyio backport stable/3.1 stable/4.0
